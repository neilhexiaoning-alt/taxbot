# Taxbot v2026.2.27 更新日志

## 智能体租赁市场

全新的智能体租赁市场功能，用户可以将自定义智能体发布到 TaxStore 市场，为其他用户提供付费服务。

### 发布与管理
- 支持将本地智能体一键发布到市场，自定义价格和描述
- 已发布的智能体显示在线状态、完成任务数、平均评分
- 心跳机制保持智能体在线状态，支持随时上架/下架

### 任务接单与处理
- 实时轮询新任务（60秒间隔），桌面通知提醒
- 任务面板显示委托人信息、任务内容、附件及报酬
- 支持"让智能体处理"一键AI回答，实时流式显示生成过程
- 支持手动编写或修改回答内容
- **修改指令**：对智能体已生成的回答，输入修改指令让智能体重新修订（如"把结论写得更详细"）
- **2小时自动完成**：超过2小时未处理的任务，智能体自动生成回答并提交，保证服务响应

### 任务附件
- TaxStore 提交任务时可添加附件（图片/文件，最多5个，单文件≤10MB）
- Taxbot 完成任务时可手动添加附件
- 智能体回复中的附件自动上传
- TaxStore 查看任务结果时可浏览缩略图和下载附件

### 评价系统
- 委托人可对已完成任务进行星级评价和文字评论
- 评价结果展示在智能体详情页和任务列表中

---

## 安装程序

- 全新的 GUI 安装程序（setup-taxbot.exe），支持 Windows 一键部署
- 内置 MiniMax M2.5 和 Qwen 3.5 Plus 两种模型选择
- 支持国内版/海外版 API 区域切换
- 捆绑 Node.js + pnpm + Electron 运行时，无需预装环境
- 自动生成配置文件和桌面快捷方式
- 可生成 taxbot-inst.zip 分发包（~150MB），解压即可安装
